Let’s first learn something that will help you in this process.

You have been working with variables that can hold a single value what if there was a way for any variable to hold more than one. Let me introduce you to lists. Yes, it is exactly like your mom’s grocery list, a sequence of items.

Let's work on an example now. On your shell type the following:

>>> list1 = [2,4,6,8]
>>> list2 = [‘Dog’,‘Cat’,‘Fish’,‘Parrot’]

In the above example you created 2 lists. To create a list you need to add the various elements inside [] and then assign it to a variable. The first list you created is a list of even numbers less than 10. Next is a list of common pets.

You know how strings work. You can draw some parallel between strings and lists. Just like strings you can access a particular element using indexing and slice a few elements of the list.

>>> list1[2]
6
>>> list2[2:]
[‘Fish’,‘Parrot’]

You can also add elements to a list using append:

>>> list2.append(‘Rabbit’)
>>> print(list2)
[‘Dog’,‘Cat’,‘Fish’,‘Parrot’,‘Rabbit’]

You can also delete elements from lists using the index or the element you want to delete.

>>> list1.pop(1)
4
>>> print(list1)
[2,6,8]
>>> list1.remove(6)
>>> print(list1)
[2,8]

Using variableName.pop(index) method to remove element at particular index and variableName.remove(element) method to delete the first occurence of the given element from the list.
