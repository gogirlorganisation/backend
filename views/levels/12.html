These computers only understand 1s and 0s.

In Python, we can think of these 1s and 0s as True and False where 1 is True and 0 is False. Thus with these we can form what is called as Boolean Expressions. Let's work on a few examples.

In the shell go ahead type the following:

>>> 4 < 5
True
>>> 5 < 4
False
>>> 5 == 5
True
>>> not False
True

Apart from the ones we used we can use <=, >=, != as well. In Python you can join two boolean expression using and and or. Before taking any other example, carefully look at the following tables.

For and:

Expression Result
True and True True
True and False False
False and True False
False and False False

For or:

Expression Result
True or True True
True or False True
False or True True
False or False False

Keeping these in mind we can form complex expression for evaluation.

Let's go to the shell.

>>> True or (False and True)
True
>>> not False or (False and True) and True
True
